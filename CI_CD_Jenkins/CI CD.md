

### what is CI CD
```
Process of automating the software delivery process from code commit to deployment 

-- instead of doing manually  -
	- developer writes code 
	  someone build code manually 
	  someone tests code manually
	  someone copies code to server manually 
	  if anything goes wrong , production fails
	  
- ci/cd makes this process automatic , repeatable , faster and safer 
   
	  

```

### problems before CI/CD
```
- manual deployment mistakes
- slow releases
- inconsistent steps
- " works on my machine " issue 
- testing skipped due to pressure 
- production issues after release 

```

### Why companies love CI/CD
```
- fewer human error 
- faster releases 
- better quality 
- repeatable deployments 
- early traceability 
- early bug detection 

```
